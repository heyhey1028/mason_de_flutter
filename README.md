# Using Mason


## Creating new brick
execute below to create brick template

```bash
cd ./bricks
mason new <your brick name>
cd <your brick name>
tree
 .
 ├── CHANGELOG.md
 ├── LICENSE
 ├── README.md
 ├── __brick__
 │   └── HELLO.md
 └── brick.yaml
```
